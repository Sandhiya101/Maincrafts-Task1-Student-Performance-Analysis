# Student Performance Analysis

## Overview

This is my Task 1 submission for the Data Science and Data Analysis with Python Internship.

In this project, I analyzed the Student Performance dataset using Python. The dataset was explored, cleaned, and analyzed to identify patterns in student academic performance. Various statistical measures and visualizations were used to gain insights into the data.

## Tools Used

* Python
* Pandas
* Matplotlib
* Google Colab

## Analysis Performed

* Dataset exploration
* Missing value checking
* Duplicate value checking
* Data cleaning
* Statistical analysis
* Data visualization

## Key Findings

* Average Final Grade (G3): 10.42
* Students Scoring Above 15: 40
* Study Time and Grade Correlation: 0.0978
* Female Average Grade: 9.97
* Male Average Grade: 10.91

## Visualizations

* Distribution of Final Grades (Histogram)
* Study Time vs Final Grade (Scatter Plot)
* Average Grade by Gender (Bar Chart)

## Conclusion

The analysis indicates that study time has a weak positive correlation with final grades. Male students achieved a slightly higher average final grade than female students in this dataset. Visualizations helped in understanding grade distribution and performance trends among students.
